Detailed Description

Arcade namespace.

This namespace is used to define all the classes and functions that will be used in the arcade. The graphical libraries and the games must use this namespace.

enum arcade::Event
strong

An enumeration for representing different events.

This enumeration is used to represent the different events that can be received by the graphical library. In order to handle the user input in the arcade, the graphical library must return one of these events.
